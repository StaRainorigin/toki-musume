# 萌系陪伴风 UI 重设计方案

## 设计理念
把冷冰冰的工具界面变成一个有温度的陪伴角色界面。暖色调、圆角、气泡有尾巴、角色头像、每个模式有自己的颜色和表情。

## 改动清单

### 1. `src/styles/variables.css` — 暖色系配色重写
- 主色从冷蓝 `#4a90d9` 改为暖粉 `#ff8fa3`
- 辅助色：薄荷绿 `#7dd3ae`、蜜橘 `#ffb085`、薰衣草 `#c4a7e7`
- 背景从纯白改为奶油色 `#fff9f5`
- 加阴影 token（柔和投影）、渐变 token
- 加字体 token：`--font-rounded: 'Segoe UI', 'Hiragino Sans', sans-serif`
- 每种模式一个主题色变量

### 2. `src/App.vue` — 加角色头像区
- header 左侧加一个圆形角色头像（用 emoji 或 CSS 绘制的简化头像，根据模式变表情）
- 角色名下方加状态文字（"正在监督你学习～" / "陪你聊天中" / "休息一下吧"）
- 背景加柔和渐变

### 3. `src/components/ChatPanel.vue` — 萌系气泡
- assistant 气泡：粉色背景 + 左下圆角尖尾
- user 气泡：薄荷绿背景 + 右下圆角尖尾
- system 消息：居中小胶囊样式
- assistant 消息前加角色小头像
- 气泡圆角加大到 18px

### 4. `src/components/StatusBar.vue` — 模式按钮彩色化
- 每个模式按钮有独立颜色：陪伴=粉、学习=蓝、工作=橙、休息=绿
- active 状态填充对应颜色，非 active 为浅色描边
- 按钮加圆角胶囊形状
- 模式名前加 emoji 图标

### 5. `src/components/MessageInput.vue` — 圆润输入框
- 输入框改为胶囊形（大圆角）
- 发送按钮改为圆形带纸飞机 emoji
- 输入框获焦时有柔和粉色光晕

### 6. `src/components/TodaySummary.vue` — 摸鱼计数器
- 摸鱼次数用彩色小徽章显示
- 加一个小鱼 emoji 🐟

### 7. `src/components/SettingsPanel.vue` — 圆角表单
- 卡片圆角加大
- 保存按钮改为主色渐变
- 整体配色跟随暖色系

### 8. `src/components/DebugPanel.vue` — 轻量主题调整
- 背景改为半透明白色毛玻璃
- 保持功能不变，只调配色跟随主题

### 9. `src/components/HistoryView.vue` — 配色跟随
- 日期列表 hover 色改为暖色
- 评论框背景改为奶油色

## 不改动的部分
- 组件 props/emits 接口不变
- composable 逻辑不变
- controller/types 不变
- Rust 后端不变
- 测试不变

- `vue-tsc --noEmit` 通过
- `pnpm test` 74 个测试通过
- 应用启动，视觉确认萌系风格生效
