# Phase 2: SetWinEventHook 替代轮询

## Rust 端

### `src-tauri/src/winapi.rs`
- 重构 `get_foreground_window()` 提取出 `foreground_info_for(hwnd)` 共享函数
- 新增 `start_foreground_hook(app: AppHandle)`:
  - spawn 专用线程
  - `SetWinEventHook(EVENT_SYSTEM_FOREGROUND, ..., WINEVENT_OUTOFCONTEXT)`
  - 线程跑 `GetMessageW` 消息循环
  - callback 里用 `foreground_info_for(hwnd)` + `app.emit("foreground_window_changed", payload)`
- 新增 `stop_foreground_hook()`: `UnhookWinEvent` + 退出线程

### `src-tauri/src/lib.rs`
- 注册命令 `start_window_hook` / `stop_window_hook`
- 在 `setup` 里自动启动 hook
