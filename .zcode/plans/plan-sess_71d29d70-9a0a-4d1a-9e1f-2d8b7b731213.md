# Iconify 图标替换 + 主题切换方案

## 1. 安装依赖
```bash
pnpm add @iconify/vue @iconify-json/tabler @iconify-json/material-symbols
```
- `@iconify/vue` — Vue 组件 `<Icon icon="..." />`
- `@iconify-json/tabler` — 离线打包 Tabler 图标集（UI 功能图标：关闭/设置/发送/保存/调试等）
- `@iconify-json/material-symbols` — 离线打包 Material Symbols 图标集（模式/角色图标：花/心/书/公文包/咖啡等）

离线模式：Tauri 桌面应用不需要联网，图标数据打包在本地。

## 2. 新增 `src/composables/useTheme.ts` — 主题切换
- 5 套预设主题：`pink`（默认粉）、`blue`（蓝）、`green`（绿）、`purple`（紫）、`dark`（暗色）
- 每套主题覆盖 `variables.css` 的 CSS 变量（通过 `document.documentElement.style.setProperty`）
- 持久化到 `localStorage`，启动时恢复
- 导出 `currentTheme` ref + `themes` 列表 + `setTheme(name)` 方法
- 在 header 加一个 🎨 主题切换下拉/按钮组

## 3. 新增 `src/composables/useModeIcons.ts` — 统一图标映射
- 把 `App.vue` 的 `modeEmoji` 和 `StatusBar.vue` 的 `modes` 合并为一个文件
- 返回 `Record<Mode, { icon: string; label: string; status: string }>`
- 例：`study` → `{ icon: 'material-symbols:menu-book-rounded', label: '学习', status: '正在监督你学习～' }`
- `ChatPanel` 的 `avatar` prop 改为 icon 名字符串
